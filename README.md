# Свадебный сайт Яна и Ани

Статический сайт, готовый для публикации через GitHub и Netlify.

## Структура
- `index.html` — главная страница
- `seating.html` — интерактивная рассадка
- `css/` — стили
- `js/` — сценарии
- `img/` — изображения

## Публикация на Netlify
Подключите репозиторий к Netlify. Build command оставьте пустым, Publish directory укажите `.`.
